# 🔧 Lopes Componentes Pneumáticos — Institutional Website

> Institutional website developed for a company specializing in industrial pneumatic components, built with pure HTML, CSS and JavaScript — no frameworks.

---

## 🌐 About the Project

This project is a complete product catalog website for **Lopes Componentes Pneumáticos**, a company based in Indaial, Santa Catarina, Brazil. The goal was to create a professional, functional and responsive website that allows customers to browse products and request quotes directly via WhatsApp.

---

## ✨ Features

- 🛒 **Quote Cart** — customers select products with specific options and send a complete quote request via WhatsApp
- 🔩 **Product Configurator** — selectors for thread sizes, coil options and voltage, with mandatory field validation
- 🖼️ **Image Carousel** — multiple product images with navigation arrows and dot indicators
- 🔍 **Image Zoom Modal** — click on any product image to open a fullscreen modal with carousel navigation
- 📱 **WhatsApp Integration** — contact form and cart checkout send messages directly to WhatsApp
- 📍 **Google Maps Embed** — store location embedded on the contact section
- 📱 **Responsive Design** — adapted for mobile, tablet and desktop

---

## 🛠️ Technologies Used

- **HTML5** — semantic structure
- **CSS3** — manual styling, no frameworks (no Tailwind, no Bootstrap)
- **JavaScript** — DOM manipulation, dynamic rendering, localStorage for cart persistence
- **Lucide Icons** — icon library
- **Google Fonts** — Roboto and Montserrat typography

# 🔧 Lopes Componentes Pneumáticos — Site Institucional

> Site institucional desenvolvido para empresa especializada em componentes pneumáticos industriais, construído com HTML, CSS e JavaScript puro — sem frameworks.

---

## 🌐 Sobre o Projeto

Este projeto é um site completo de catálogo de produtos para a **Lopes Componentes Pneumáticos**, empresa localizada em Indaial, Santa Catarina. O objetivo foi criar um site profissional, funcional e responsivo que permite aos clientes navegar pelos produtos e solicitar orçamentos diretamente pelo WhatsApp.

---

## ✨ Funcionalidades

- 🛒 **Carrinho de Orçamento** — o cliente seleciona produtos com opções específicas e envia uma solicitação de orçamento completa via WhatsApp
- 🔩 **Configurador de Produtos** — seletores de rosca, opção de bobina e voltagem, com validação de campos obrigatórios
- 🖼️ **Carrossel de Imagens** — múltiplas imagens por produto com setas de navegação e bolinhas indicadoras
- 🔍 **Modal de Zoom** — clique em qualquer imagem para abrir um modal em tela cheia com navegação por carrossel
- 📱 **Integração com WhatsApp** — formulário de contato e finalização do carrinho enviam mensagens diretamente ao WhatsApp
- 📍 **Google Maps** — localização da loja incorporada na seção de contato
- 📱 **Design Responsivo** — adaptado para mobile, tablet e desktop

---

## 🛠️ Tecnologias Utilizadas

- **HTML5** — estrutura semântica
- **CSS3** — estilização manual, sem frameworks (sem Tailwind, sem Bootstrap)
- **JavaScript** — manipulação do DOM, renderização dinâmica, localStorage para persistência do carrinho
- **Lucide Icons** — biblioteca de ícones
- **Google Fonts** — tipografia Roboto e Montserrat
